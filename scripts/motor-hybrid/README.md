# Motor Hybrid - Build Scripts

## Scripts Disponíveis

| Script | Descrição |
|--------|-----------|
| `build-motor.sh` | Compila e valida Motor Hybrid |
| `test-motor.sh` | Executa testes unitários (futuro) |

## Uso
```bash
# Build
bash ~/shaka-api/scripts/motor-hybrid/build-motor.sh

# Testes (quando implementados)
bash ~/shaka-api/scripts/motor-hybrid/test-motor.sh
```
